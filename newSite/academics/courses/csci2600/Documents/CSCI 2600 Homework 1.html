Problems

Problem 1 (6 pts, 1 pt each): Condition Strength

Indicate the weakest condition in each set. Write "None" if the conditions are unrelated by implication. Unless otherwise stated, assume all variables are ints.

1. { x is odd && y = x + 2 }
   { x is odd && y is odd }
   { x % 2 ≠ 0 && y ≠ 2 }

 

2. { 0 ≤ x ≤ 5 }
   { 0 ≤ x < 5 }
   { 1 ≤ x < 5 }

 

3. { x > 0 && y ≥ 0 }
   { x > 10 || y = 0 }

 

4. { x is divisible by 6 }
   { the rightmost digit of x is even }
   { the sum of digits of x is divisible by 3 }


5. { -5 < x < 40 }
   { x = 25 || x = 35 }
   { 0 < x < 30 }

 

6. assume "result" is a double
   { abs(result * result - x) ≤ 0.0001 }
   { abs(result * result - x) ≤ -0.000001 }
