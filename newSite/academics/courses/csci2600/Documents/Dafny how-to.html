To help those of you who wanted to use a standalone version of Dafny (as opposed to a Web page front-end which can get slow or unresponsive) on a Windows system but were having some issues, here is a short tutorial.

A mini how-to Using Dafny in Windows


Monospaced font denotes the characters that you need to type in literally, file names/paths, UI control names, or the output of your commands.

1) Navigate to https://github.com/Microsoft/dafny/releases
2) Download the latest Dafny release for Windows (as of the time of writing, it is 3.1.0, i.e., the file to download is dafny-3.1.0-x64-win.zip).
3) Right-click on the downloaded ZIP file and select Properties.
4) In the lower right corner of the Properties window you should see an unchecked Unblock checkbox.
5) Check this box and click OK. (Once checked, this box disappears from the Properties window; so if you open up the Properties again, you will no longer see that box; this is normal.) Checking this box tells Windows that you trust the source from which you acquired this ZIP file and you allow Windows to run this software.
6) Unpack the contents of the ZIP file to a folder on your local machine.
7) Launch the command prompt or a PowerShell window. As usual, there are multiple ways of doing things. For instance, you could press Win+R on your keyboard and then type in cmd in the Open field of the dialog box that appears and then click OK. (Here's a link to the list which includes this and some other useful hotkeys in Windows: https://support.microsoft.com/en-ca/help/12445/windows-keyboard-shortcuts )
8) Change the directory to the folder where you extracted the contents of the ZIP file. You need to type cd folder_name where folder_name is the name of the folder you want to change to. Please note that if folder_name contains spaces, you need to enclose the entire folder_name in quotes.
9) Type in a command to launch Dafny.exe giving the file name with your Dafny program as a parameter. (Please note that Dafny would expect the file to have a ".dfy" extension.)

Here's an example of how this works.

Downloaded the ZIP and extracted it into c:\Program Files\dafny\. Launched the command prompt and typed cd "c:\Program Files\dafny\". Created a simple Dafny example in file c:\CSCI-2600\Lab01\Lab01.dfy. The contents of the file is as follows:

method DivisionByZero(x: int) returns (y: int)
  requires x == 1 || x == -2
  ensures y == 0
{
  y := x + 4;
  if (x > 0) {
    y := x*x - 1;
  } else {
    y := y + x;
  }
}

Verified myDafny program by typing the command, like that:

c:\Program Files\dafny>Dafny c:\CSCI-2600\Lab01\Lab01.dfy
Dafny 3.1.0.30421

Dafny program verifier finished with 1 verified, 0 errors
Compiled assembly into Lab01.dll

That's it! Hopefully this how-to would be helpful to at least some of you.
